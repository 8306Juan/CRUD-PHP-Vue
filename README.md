# php-vue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

## Requeriments
```
To use this repository it is necessary:
to use a local server such as **XAMP or WAMP**.
```

## Configuration
```
1- DB Connection
	-in: **src/php/conexion.php**

2- Axios Requests
	-in: **src/views/Productos.vue** AND **src/views/Agregar.vue**
	-Change the content of the variable **"urlPHP"** for that of your php files
```

npm run build
